# Exame 102-500 — Visão Geral

O **Exame 102** cobre tópicos avançados de administração Linux, incluindo shell scripting, gerenciamento de usuários, serviços de sistema, redes e segurança.

## Tópicos do Exame 102

| Tópico | Descrição | Peso |
|--------|-----------|------|
| [105 — Shell e Shell Scripting](topico-105-shell/README.md) | Ambiente shell, scripts bash | 8 pontos |
| [106 — Interfaces de Usuário](topico-106-interfaces/README.md) | X11, desktops gráficos, acessibilidade | 4 pontos |
| [107 — Tarefas Administrativas](topico-107-administrativo/README.md) | Usuários, grupos, agendamento, localização | 12 pontos |
| [108 — Serviços do Sistema](topico-108-servicos/README.md) | Tempo, logs, MTA, impressão | 12 pontos |
| [109 — Fundamentos de Redes](topico-109-redes/README.md) | Protocolos TCP/IP, configuração de rede, DNS | 14 pontos |
| [110 — Segurança](topico-110-seguranca/README.md) | Tarefas de segurança, GPG, SSH | 10 pontos |

## Como Estudar

1. Certifique-se de ter dominado o conteúdo do [Exame 101](../exam-101/README.md) antes de começar.
2. Leia o conteúdo de cada tópico e pratique os comandos.
3. Complete os exercícios disponíveis em cada pasta.
4. Simule o ambiente de exame com as questões de múltipla escolha.
