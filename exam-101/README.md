# Exame 101-500 — Visão Geral

O **Exame 101** cobre os fundamentos da administração de sistemas Linux, incluindo arquitetura de hardware, instalação, gerenciamento de pacotes, comandos essenciais e sistemas de arquivos.

## Tópicos do Exame 101

| Tópico | Descrição | Peso |
|--------|-----------|------|
| [101 — Arquitetura do Sistema](topico-101-arquitetura/README.md) | Configuração de hardware, boot, runlevels | 6 pontos |
| [102 — Instalação e Pacotes](topico-102-instalacao/README.md) | Particionamento, bootloader, pacotes DEB/RPM | 8 pontos |
| [103 — Comandos GNU e Unix](topico-103-comandos/README.md) | Linha de comando, filtros, processos, redirecionamento | 26 pontos |
| [104 — Dispositivos e Filesystems](topico-104-filesystems/README.md) | Partições, montagem, permissões, links | 20 pontos |

## Como Estudar

1. Leia o conteúdo de cada tópico na ordem acima.
2. Pratique os comandos em um sistema Linux real ou virtual.
3. Complete os exercícios disponíveis em cada pasta.
4. Revise os pontos-chave antes de avançar.

## Distribuições Recomendadas para Prática

- **Ubuntu** / **Debian** (sistemas baseados em Debian — para praticar `apt` e `dpkg`)
- **CentOS** / **Fedora** / **openSUSE** (sistemas baseados em Red Hat — para praticar `rpm` e `yum`/`dnf`)
