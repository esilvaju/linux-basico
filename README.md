# 🐧 Linux Básico — Guia de Estudos LPIC-1

Bem-vindo ao repositório **Linux Básico**! Este espaço foi criado para estudantes que estão aprendendo Linux e desejam se preparar para a certificação **LPIC-1** do [Linux Professional Institute (LPI)](https://www.lpi.org/pt-br/our-certifications/lpic-1-overview/).

---

## 📋 O que é o LPIC-1?

O **LPIC-1** é a primeira certificação de administrador Linux do LPI. Ela valida a capacidade de:

- Realizar tarefas de manutenção na linha de comando
- Instalar e configurar um computador com Linux
- Configurar redes básicas

A certificação é composta por **dois exames**:

| Exame | Código | Conteúdo |
|-------|--------|----------|
| Exame 1 | 101-500 | Arquitetura do sistema, instalação, pacotes, comandos GNU/Unix, filesystems |
| Exame 2 | 102-500 | Shell scripting, interfaces, tarefas administrativas, redes, segurança |

## 🚀 Como Usar Este Repositório

1. **Clone o repositório**
   ```bash
   git clone https://github.com/esilvaju/linux-basico.git
   cd linux-basico
   ```

2. **Escolha um tópico** na trilha de estudos acima e leia o conteúdo.

3. **Pratique** com os exercícios disponíveis em cada tópico (`exercicios.md`).

4. **Experimente** os comandos em uma distribuição Linux (Ubuntu, Debian, CentOS, etc.) ou em uma máquina virtual.

## 🎯 Pré-requisitos

- Nenhum conhecimento prévio de Linux é necessário para começar
- Acesso a um sistema Linux (físico, virtual ou WSL no Windows)
- Vontade de aprender! 💪
