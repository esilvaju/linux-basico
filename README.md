# Linux Básico — Preparatório LPIC-1

Repositório de estudos para a certificação **LPIC-1** (Linux Professional Institute Certification – Nível 1), oferecida pelo [LPI (Linux Professional Institute)](https://www.lpi.org/pt-br/our-certifications/lpic-1-overview/).

---

## 📋 Sobre a Certificação LPIC-1

O **LPIC-1** é o primeiro nível do programa de certificação profissional de múltiplos níveis do LPI. Ele valida a capacidade do candidato de realizar tarefas de manutenção na linha de comando, instalar e configurar um computador com Linux, configurar rede básica e realizar operações de administração de sistemas.

A certificação é **distribuição-neutra** e amplamente reconhecida pela indústria de TI em todo o mundo.

### Objetivos da Certificação

- Trabalhar na linha de comando Linux
- Realizar tarefas básicas de manutenção em sistemas Linux
- Instalar e configurar computadores rodando Linux
- Configurar rede básica
- Usar e gerenciar pacotes de software

---

## 📝 Exames

A certificação LPIC-1 é composta por **dois exames**:

| Exame | Código | Descrição |
|-------|--------|-----------|
| Exame 1 | **101-500** | Cobre tópicos de arquitetura do sistema, instalação e gerenciamento de pacotes, comandos GNU e Unix, e sistema de arquivos |
| Exame 2 | **102-500** | Cobre tópicos de shells e scripts, interfaces de usuário, tarefas administrativas, serviços essenciais do sistema, fundamentos de rede e segurança |

Ambos os exames têm **60 questões** e uma duração de **90 minutos** cada. A nota mínima para aprovação é **500 pontos** (em uma escala de 200 a 800).

## 📚 Conteúdo Programático

### Exame 101-500

#### Tópico 101 – Arquitetura do Sistema
- 101.1 Identificar e editar configurações de hardware
- 101.2 Inicializar o sistema
- 101.3 Mudar runlevels / boot targets e desligar ou reinicializar o sistema

#### Tópico 102 – Instalação e Gerenciamento de Pacotes do Linux
- 102.1 Design do layout do disco rígido
- 102.2 Instalar um gerenciador de boot
- 102.3 Gerenciar bibliotecas compartilhadas
- 102.4 Usar o gerenciamento de pacotes Debian
- 102.5 Usar o gerenciamento de pacotes RPM e YUM
- 102.6 Linux como sistema operacional virtualizado

#### Tópico 103 – Comandos GNU e Unix
- 103.1 Trabalhar na linha de comando
- 103.2 Processar streams de texto usando filtros
- 103.3 Gerenciamento básico de arquivos
- 103.4 Usar streams, pipes e redirecionamentos
- 103.5 Criar, monitorar e matar processos
- 103.6 Modificar prioridades de execução de processos
- 103.7 Buscar em arquivos de texto usando expressões regulares
- 103.8 Operações básicas de edição de arquivos usando vi

#### Tópico 104 – Dispositivos, Sistemas de Arquivos Linux e o Filesystem Hierarchy Standard
- 104.1 Criar partições e sistemas de arquivos
- 104.2 Manter a integridade de sistemas de arquivos
- 104.3 Controle de montagem e desmontagem de sistemas de arquivos
- 104.5 Gerenciar permissões e propriedades de arquivos
- 104.6 Criar e alterar links físicos e simbólicos
- 104.7 Localizar arquivos do sistema e colocar arquivos nos locais corretos

### Exame 102-500

#### Tópico 105 – Shells e Scripts
- 105.1 Personalizar e usar o ambiente shell
- 105.2 Personalizar ou escrever scripts simples

#### Tópico 106 – Interfaces de Usuário e Desktops
- 106.1 Instalar e configurar o X11
- 106.2 Configuração gráfica de desktops
- 106.3 Acessibilidade

#### Tópico 107 – Tarefas Administrativas

- 108.4 Gerenciar impressoras e impressão

- 109.1 Fundamentos de protocolos de internet
- 109.2 Configuração básica de rede
- 109.3 Solução básica de problemas de rede
- 109.4 Configurar DNS do lado do cliente

#### Tópico 110 – Segurança
- 110.1 Realizar tarefas de administração de segurança
- 110.2 Configurar segurança do host
- 110.3 Protegendo dados com criptografia

## 🎯 Pré-requisitos

Não há pré-requisito formal para realizar os exames do LPIC-1. No entanto, recomenda-se que o candidato:

- Tenha familiaridade básica com computadores
- Possua alguma experiência prévia com Linux ou sistemas Unix-like
