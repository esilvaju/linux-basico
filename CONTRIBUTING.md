# 🤝 Guia de Contribuição

Obrigado por querer contribuir com o **Linux Básico**! Este repositório existe para ajudar estudantes a aprenderem Linux com base no currículo do LPIC-1.

---

## Como Contribuir

### 1. Faça um Fork do Repositório

Clique no botão **Fork** no canto superior direito da página do repositório no GitHub.

### 2. Clone o Seu Fork

```bash
git clone https://github.com/SEU_USUARIO/linux-basico.git
cd linux-basico
```

### 3. Crie uma Branch para Sua Contribuição

```bash
git checkout -b minha-contribuicao
```

### 4. Faça as Suas Alterações

- Siga a estrutura de diretórios existente.
- Escreva o conteúdo em **Português**.
- Use linguagem clara e acessível para iniciantes.
- Sempre que possível, adicione exemplos práticos de comandos.

### 5. Faça o Commit das Suas Alterações

```bash
git add .
git commit -m "Adiciona: [descrição breve da sua contribuição]"
```

### 6. Envie Sua Branch

```bash
git push origin minha-contribuicao
```

### 7. Abra um Pull Request

No GitHub, abra um **Pull Request** da sua branch para a branch `main` do repositório original.

---

## 📋 Diretrizes de Conteúdo

- **Linguagem:** Português (Brasil)
- **Nível:** Iniciante a intermediário (foco em LPIC-1)
- **Formato:** Markdown (`.md`)
- **Estrutura por tópico:**
  - `README.md` — Teoria, explicações e exemplos
  - `exercicios.md` — Exercícios práticos e questões de fixação

### Padrão para Arquivos de Tópico

Cada arquivo `README.md` de tópico deve conter:

1. Título e descrição do tópico
2. Objetivos de aprendizagem (alinhados ao LPIC-1)
3. Conteúdo teórico com exemplos de comandos
4. Resumo / pontos-chave

Cada arquivo `exercicios.md` deve conter:

1. Exercícios práticos numerados
2. Questões de múltipla escolha (no estilo do exame LPIC-1)
3. Respostas comentadas (no final ou em seção separada)
